## Restaurant Food Website Platform

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/6b175c56f45a4143b228cdbad5e01037/ghb20250919) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can be used to run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Feature Description
The system is divided into (roles): regular users, restaurants, administrators.

**Regular User Features:**
1. Login
2. Registration
3. Homepage
4. Restaurant Information
5. Food Information
6. Food Culture
7. Announcement News
8. Background Management
9. Shopping Cart
10. Personal Center
11. My Orders
12. My Addresses
13. My Favorites


**Restaurant Features:**
1. Login
2. Registration
3. Change Password
4. Personal Information
5. Restaurant Information Management
6. Food Information Management
7. Order Management



**Administrator Features:**
1. Login
2. Personal Center (Change Password, Personal Information)
3. Restaurant Management
4. User Management
5. Restaurant Information Management
6. Cuisine Category Management
7. Food Information Management
8. Food Culture Management
9. System Management (About Us, System Introduction, Carousel Image Management, Announcement News)
10. Order Management
